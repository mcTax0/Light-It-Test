# Light IT Test

### Требования

1. Наличие браузера
2. Наличие интернета 

### Описание файлов

* `index.html` - необходимый файл для запуска программы
* `main.js` - основной файл с кодом программы
* `cards.js` - файл с исходными данными

### Использование

Для тестрирования программы, Вам необходимо открыть в любом браузере файл 'index.html'. Браузер попросит Вас ввести количество игроков в данной партии,после этого нужно перейти в консоль разработчика. Для этого кликаем правой кнопкой мыши на свободную часть экрана и выбираем 'Просмотреть код', затем вкладку 'Console',где и отображается результат игры. Первая строка - козырь в данной партии, после отображается двумерный массив, внутри которого - рука каждого игрока, а в последней строке - рука с самыми сильными картами,с учётом козырей.
